Welcome to the Online Experiment

The task is about selecting design techniques for a scenario. Before reading the task, please have a look at the following introduction of the way of using the website.

The task is presented to you before the webpage for selecting design techniques appear.

Please click on Show the Instruction

In this area, there are some categories which can help you to narrow down the scope when selecting design techniques.

In this area, there are 50 design techniques. All design techniques are presented in a card format.

Please just click on “+” and the design technique will be selected. The selected design techniques will appear at the bottom-right of the webpage.

You should select three design techniques for the task.

The task will appear in this area as a reminder.

Before you come to the web page for selecting design techniques, the description of the task was already presented.

The selected design techniques will appear in this area.
